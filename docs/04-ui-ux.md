# Fase 4 - UI/UX MVP

## Objetivos

La interfaz debe transmitir confianza, seguridad, profesionalismo, simplicidad y rapidez.

## Navegacion

Visitante:

- Inicio
- Nosotros
- Servicios
- Requisitos
- FAQ
- Contacto
- Registro
- Login
- Recuperar contrasena
- Politica
- Terminos

Cliente:

- Completar perfil
- Perfil
- Mis solicitudes
- Nueva solicitud
- Detalle solicitud

Admin:

- Solicitudes
- Detalle solicitud
- Clientes

## Sistema visual

- Primario: `#1E3A8A`
- Secundario: `#2563EB`
- Fondo: `#F8FAFC`
- Texto: `#0F172A`
- Exito: `#16A34A`
- Advertencia: `#F59E0B`
- Error: `#DC2626`

## Componentes

Header, footer, sidebar, card, tabla, badge de estado, input, select, textarea, upload, modal, toast, breadcrumb, paginacion, acordeon, spinner y empty state.

## Responsive

Mobile first para telefono, tablet, laptop y desktop.
